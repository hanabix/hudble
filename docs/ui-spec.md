# UI 设计规范

本文提炼之[Rokid YodaOS-Sprite 设计规范](https://custom.rokid.com/prod/rokid_web/57e35cd3ae294d16b1b8fc8dcbb1b7c7/pc/cn/2786298057084a82b170bf725aef6b5d.html?documentId=d0db23be629d4b09be197bd0dd931298)，仅记录图片中可以明确识别的规范。图片未给出精确数值的内容，不在此文档中补充推断值。

## 适用范围

- 平台风格：Rokid `YodaOS-Sprite`
- 设计画布：`480 x 640 px`
- 最佳显示区域：`480 x 400 px`

## 字体规范

### 字号层级

| 层级 | 字号 | 行高 | 字重 |
| --- | --- | --- | --- |
| 一级 | 32px | 40px | Regular / Medium |
| 二级 | 24px | 32px | Regular / Medium |
| 三级 | 20px | 26px | Regular / Medium |
| 四级 | 18px | 24px | Regular / Medium |
| 五级 | 16px | 22px | Regular / Medium |

## 颜色与描边

### 主色

- 设计稿色值：`#40FF5E`
- 透明度：`100%`，设计稿标注为“建议”

### 使用限制

- 禁用渐变
- 禁用大面积高亮色
- 设计稿说明：低亮度下大面积纯色块容易出现显示异常，需避免大面积铺色

### 描边规则

- 描边最小值：`>= 1.5px`
- 设计稿示例描边：`1.5px`、`2px`、`4px`
- 常用圆角：`12px`

### 描边状态

| 状态 | 建议表现 |
| --- | --- |
| 常态 | 描边 40% |
| 选中 | 描边 80% |
| 按下 | 描边 100% |

## 布局规范

### 画布与安全区

- 画布大小：`480 x 640 px`
- 最佳显示区域：`480 x 400 px`
- 左边距：贴边 `0px`
- 右边距：贴边 `0px`

### 不建议展示区域

- 顶部 `160px`：显示效果不佳，慎用
- 底部 `80px`：显示效果不佳，慎用

## Icon 规范

### 风格

- 使用线性图标
- 设计目标：识别清晰、更加省电

### 推荐尺寸

| 用途 | 尺寸 |
| --- | --- |
| 应用图标 / 大图标 | 40 x 40 px |
| 常规图标 | 20 x 20 px |
| 最小图标 | 16 x 16 px |

### 双色图标

- 设计稿建议：`100%` + `50%` 两档亮度组合

### 绘制约束

- 图标设计需贴合像素格

## 落地建议

- 优先使用线性图标、描边控件和小面积高亮色，避免实心大色块
- 文字、图标和关键状态尽量放在 `480 x 400 px` 最佳显示区域内
- 若后续拿到源设计文件，应补充颜色透明度定义、图标栅格、间距 token 与组件尺寸 token
